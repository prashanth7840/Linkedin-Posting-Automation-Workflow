# LinkedIn Posting Automation Workflow

A fully automated n8n workflow that drafts LinkedIn posts using AI, sends them for human review via email, optionally generates AI images, and publishes to LinkedIn — all on a daily schedule.

Built as part of the Outskill AI Generalist Fellowship (Cohort 7, Week 9).

---

## What It Does

Every day at 10 AM, this workflow:

1. Pulls one "Ready" post idea from a Google Sheet
2. Drafts a LinkedIn post using GPT-5-mini with a CoSTAR ghostwriter prompt
3. Emails the draft to me with Approve / Revise / Cancel options
4. If I click Revise, it regenerates the post using my feedback
5. Optionally generates an image using Gemini Nano Banana Pro
6. Publishes the final post directly to LinkedIn
7. Updates the Google Sheet with the status and LinkedIn Post ID

**Result:** Three of my recent LinkedIn posts were 100% generated by this workflow. I just clicked "Approve" in my email.

## The 15 Nodes

| # | Node | Purpose |
|---|---|---|
| 1 | Schedule Trigger | Runs daily at 10 AM |
| 2 | Pick one post idea | Pulls one row from Google Sheet where Status = "Ready" |
| 3 | OpenAI Chat Model | GPT-5-mini for post generation |
| 4 | Basic LLM Chain | Generates post using ghostwriter prompt |
| 5 | Edit Fields | Shapes data for downstream nodes |
| 6 | Send for Human Review | Emails draft with approval form |
| 7 | Check Human Response | Routes based on my answer |
| 8 | OpenAI Chat Model 1 | GPT-5-mini for revisions |
| 9 | Regenerate Post Content | Regenerates using my feedback |
| 10 | Update to Cancelled | Marks sheet as cancelled |
| 11 | Needs Image? | Checks image column |
| 12 | Generate an Image | Gemini Nano Banana Pro |
| 13 | Create Image + Text Post | Publishes to LinkedIn with image |
| 14 | Create Text Post | Publishes text-only to LinkedIn |
| 15 | Update to Completed | Saves Post ID to sheet |

## Google Sheet Structure

| Column | Description |
|---|---|
| Post Idea | The topic or hook |
| Instructions | How the AI should write the post |
| Generate Image (Yes/No) | Whether to create an AI image |
| Status | Ready / Completed / Cancelled |
| Post ID | LinkedIn URN after publishing |

## Tech Stack

- **Automation Platform:** n8n (self-hosted on DigitalOcean VPS)
- **LLM:** OpenAI GPT-5-mini
- **Image Generation:** Google Gemini Nano Banana Pro
- **Data Source:** Google Sheets
- **Publishing:** LinkedIn API via OAuth2
- **Review Channel:** Gmail (sendAndWait with custom form)

## Key Design Decisions

**Human-in-the-Loop.** I never wanted AI posting without my review. The Gmail approval step keeps me in control while automating the hard parts.

**Revision loop.** If I don't like the draft, I can send feedback and the workflow regenerates using a separate "Revised Post Generator" prompt — not a full restart.

**Self-hosted.** Running on my own DigitalOcean VPS gives me full control over credentials and data. No third-party n8n cloud dependency.

**CoSTAR ghostwriter prompt.** The system prompt enforces a specific structure (Hook → Re-Hook → Meat → Insight → CTA → hashtags), banned AI-isms, and a 1300-character limit.

## How to Use

1. Set up an n8n instance (cloud or self-hosted)
2. Import the workflow JSON
3. Add credentials: OpenAI API, Google Sheets OAuth, Gmail OAuth, Google Gemini API, LinkedIn OAuth2
4. Create a Google Sheet with the columns above
5. Add post ideas with Status = "Ready"
6. Activate the workflow
